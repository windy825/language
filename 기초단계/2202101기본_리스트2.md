
### 검색

- 원하는 항목 탐색, 목적하는 탐색키가 중요

- 순차검색

  ```
  1. 정렬된 경우
  	동일함.
  
  2. 정렬되지 않은 경우
  	첫 원소부터 순회하며 체크, 끝에 이르면 완료
  	시간이 빠를수도 있고, 느릴 수도 있음
  	찾고자 하는 원소의 인덱스에 따라 검색시간이 달라짐
  	시간 복잡도는 O(n)
  	
  
  ```

- 이진 검색

  ```
  효율적인 검색 방법
  가운데 항목의 키값과 비교하여, 다음 검색 위치를 정하고 검색함 (중간값 탐색)
  
  but 자료가 정렬되어 있어야 적용할 수 있음
  
  시간 복잡도 : O(logN)
  ```
